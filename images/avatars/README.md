# 卡通头像资源设计规范

## 需求说明
需要设计30个卡通风格的头像，用于用户匿名注册系统。当前使用emoji作为临时头像，需要替换为真实的卡通图像。

## 文件规范
- **数量**: 30个
- **格式**: PNG（支持透明背景）
- **尺寸**: 100×100像素（建议设计为200×200像素，然后缩放以保证清晰度）
- **命名**: `0.png`, `1.png`, `2.png`, ..., `29.png`
- **位置**: `/images/avatars/` 目录

## 设计主题建议
30个头像可以包含以下主题（每个主题3-5个变体）：

### 动物系列（15个）
- 可爱哺乳动物：熊猫、狐狸、海豚、猫咪、兔子、松鼠、刺猬、小鹿
- 鸟类：小鸟、企鹅、猫头鹰、鸭子
- 其他：青蛙、猴子、狮子

### 奇幻系列（10个）
- 神话生物：独角兽、龙、凤凰、小精灵
- 太空主题：外星人、机器人、星球、火箭
- 自然主题：花朵、树木、太阳、月亮

### 职业/角色系列（5个）
- 学生、老师、医生、科学家、艺术家

## 设计风格要求
1. **统一风格**: 所有头像采用相同的卡通绘制风格
2. **色彩明亮**: 适合儿童审美，色彩鲜艳明快
3. **表情友好**: 表情积极、友好、可爱
4. **简洁清晰**: 避免过于复杂细节，在100×100像素下清晰可辨
5. **性别中立**: 尽量避免明显的性别特征，适合所有用户

## 技术规范
1. **背景透明**: PNG格式支持alpha通道，头像主体外区域透明
2. **文件大小**: 每个文件控制在10KB以内
3. **色彩模式**: RGB
4. **分辨率**: 72-150 DPI
